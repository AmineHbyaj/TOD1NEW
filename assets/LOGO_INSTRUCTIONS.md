# Instructions pour le logo TOD

## Fichier requis

Placez le logo TOD dans ce dossier avec le nom exact : **`tod-logo.png`**

## Spécifications recommandées

- **Format** : PNG (avec transparence de préférence)
- **Taille recommandée** : 400x200px (ou ratio 2:1)
- **Couleur** : Logo TOD en couleur dorée/or sur fond transparent
- **Poids** : < 100KB pour un chargement optimal

## Emplacement

Le logo doit être placé à : `assets/tod-logo.png`

Les fichiers HTML référencent le logo avec le chemin relatif : `assets/tod-logo.png`

## Alternative

Si vous souhaitez utiliser un autre format ou nom de fichier, modifiez les références dans :
- `loader.html` (ligne 17)
- `error.html` (ligne 17)
